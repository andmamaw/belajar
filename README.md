# MAMAW Portfolio Admin Dashboard

Aplikasi portfolio full-stack berbasis Laravel dan Supabase PostgreSQL.  
Project ini memiliki halaman portfolio publik dan admin panel untuk mengelola konten secara dinamis.

## Fitur Utama

- Portfolio publik dinamis
- Admin login
- CRUD Profile
- CRUD Project
- CRUD Services
- CRUD Experience
- CRUD Education
- Dashboard ringkasan data
- Login activity tracking
- Popup sukses dan gagal
- Auto logout setelah 30 menit tidak aktif
- Custom 404 dan 403 page
- Tema dashboard modern dengan logo dan favicon custom

## Tech Stack

- Laravel
- Blade
- Tailwind CSS
- Supabase PostgreSQL
- Laravel Breeze
- Vite
